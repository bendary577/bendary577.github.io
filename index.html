ETeach Web App
Back End

ETeach is an e-learning platform that aims to make the teacher's communication with his students easier and more flexible. It also aims to spread the philosophy of distance e-learning. It allows the teacher to publish his educational clips to be available to students. It provides a mechanism for testing and evaluating students. Which makes the educational process easier for the student and the teacher. I programmed the site's backend in Java using the popular Spring framework and Mysql database. My Easy Web App
Front End

My Easy is an e-commerce platform and an electronic payment service at the same time. It allows the user to create a store to sell and promote his products, create his own project, and start earning and withdrawing money. It is also an electronic payment service that allows the user to create invoices and send them to the concerned persons to pay the bill through the MyEasy payment gateway. I developed the front end of MyEasy application using JavaScript and the famous ReactJS framework and using HTML, CSS and Bootstrap web technologies.
